# Türkçe Nefret Söylemi Veri Seti 2
10,224 adet Türkçe tweetten oluşan etiketlenmiş nefret söylemi veri setidir.

Veri setindeki sınıf dağılımı şöyledir:

| Etiket  | Tweet Sayısı |
| ------------- | ------------- |
| nefret   | 2.336  |
| saldırgan   | 166  |

| **TOPLAM**  | **10.224**  |
